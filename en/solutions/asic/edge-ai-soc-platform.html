PLATFORM ARCHITECTURE

Platform Architecture

Modular design, hardware-software co-optimization, and a flexible, configurable edge AI core architecture

This platform adopts a modular architecture design, divided into Software Layer and Hardware Layer:

Software Layer includes the model compiler, SDK, and Runtime environment; Hardware Layer integrates CIM AI IP, Tensor Engine, Memory Subsystem, Control / NoC, and Peripheral IPs.

Through standardized interfaces and flexible module configurations, it can be freely customized according to application requirements.
