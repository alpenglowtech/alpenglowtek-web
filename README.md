## 本地開發

1. 克隆專案
```bash
git clone [repository-url]
cd 0-06-公司網站
```

2. 開啟網站
```bash
# 使用任何本地伺服器，例如：
python -m http.server 8000
# 或
npx serve .
```

3. 在瀏覽器中開啟 `http://localhost:8000`
